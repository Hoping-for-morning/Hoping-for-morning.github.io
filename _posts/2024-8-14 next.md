# Next 设置环境变量

## Next 设置环境变量

### 公有环境变量

以 `NEXT_PUBLIC_` 前缀开头的环境变量会被注入到前端代码中。这些变量在浏览器中可用。

```shell
NEXT_PUBLIC_API_URL=https://api.example.com
```



### 私有环境变量

没有 `NEXT_PUBLIC_` 前缀的环境变量只会在服务器端可用。这些变量不会被注入到浏览器中。

```
SECRET_API_KEY = your_secret_api_key
```
